Vault Ledger - User Guide

This guide explains all features of your Vault Ledger, a secure and offline-first daily income & expense tracker.

Add Entry

  • Select a date (defaults to today).
  • Enter a description (e.g., "Salary", "Groceries").
  • Type the amount.
  • Choose Income or Expense.
  • Click "Add / Update" to save the record.

Edit Entry

  • Click the Edit button next to a transaction.
  • The form fills automatically with that record’s data.
  • Make your changes and press "Add / Update".

Delete Entry

  • Click Delete beside any transaction.
  • The entry is removed, and balances recalculate instantly.

Import / Export

  • Import: Load a `.json` file to restore your ledger.
  • Export: Save data in JSON, Excel, or PDF.
  • Custom filenames supported when exporting.
  • PDF reports include only transaction details (no action buttons).

Balance Summary

A live summary card shows your Total Income, Total Expenses, and Current Balance.

Undo / Redo

  • Undo reverts your last change.
  • Redo reapplies the undone change.

Closing Balance

Every row shows a closing balance after that transaction, so you always see your exact running balance.

Charts & Visuals

  • Interactive Pie, Bar, and Line charts.
  • Charts update live as you add or edit transactions.
  • Downloadable as PNG.

Filters

  • Filter transactions by Date Range, Type, or Keyword.
  • Clear filters anytime to reset view.

Insights

  • Highest Income: Shows amount, description, and date.
  • Highest Expense: Shows amount, description, and date.
  • Zero Spent Days: Lists days without expenses.
  • Income Ratio: Expense-to-Income % calculation.

Theme & Dark Mode

  • Switch between 20+ themes (light, dark, neon, sunset, cyberpunk, etc.).
  • Uses CSS variables for consistent styling.
  • Charts & tables adapt automatically to theme.
