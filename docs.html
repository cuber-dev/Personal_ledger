📘 Expense Tracker - User Guide

This page explains how to use the daily income & expense tracker effectively.

Quick Links: Back to Home Add Entry Edit Entry Delete Entry Import/Export View Balance

➕ Add Entry

  • Choose a date (defaults to today).
  • Enter a description (e.g., "Bus fare", "Received from Dad").
  • Input the amount.
  • Select either Income or Expense.
  • Click "Add / Update" to save the record.

✏️ Edit Entry

  • Click the Edit button beside the transaction.
  • The form above will auto-fill with that data.
  • Make changes and click "Add / Update".

🗑️ Delete Entry

  • Click the Delete button beside the entry to remove it permanently.

📁 Import / Export

  • Use the file input button to import a JSON file containing saved records.
  • Use the "Export JSON" button to download your current data.

💰 View Balance

The balance section at the top shows your current net amount: All income - All expenses.
