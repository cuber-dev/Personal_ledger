Add Entry

  • Choose a date (defaults to today automatically).
  • Enter a description (e.g., "Bus fare", "Salary").
  • Input the amount.
  • Select Income or Expense.
  • Click "Add / Update" to save the record.

Edit Entry

  • Click the Edit button beside any transaction.
  • The form auto-fills with the selected data.
  • Make necessary changes and click "Add / Update".

Delete Entry

  • Click the Delete button beside any transaction.
  • This will permanently remove the record from your ledger.

Import / Export

  • Import: Click the file input to load a `.json` file and continue from a previous session.
  • Export: Choose between JSON, Excel, or PDF formats for backup or sharing.
  • You can also give a custom filename before exporting.

View Balance

A summary card at the top shows your total Income, Expenses, and Current Balance.

Undo / Redo

  • Click the Undo button to revert your last change.
  • Click Redo to bring it back again.
  • This helps prevent accidental deletions or changes.

Closing Balance

  • Each row displays the closing balance after that transaction.
  • This gives you a real-time snapshot of your funds after every entry.

Visual Charts

  • Scroll down to see bar, pie, and line charts representing your financial activity.
  • Charts auto-update as you add, edit, or delete entries.

Special Insights

  • Highest Income: Displays the largest income with description & date.
  • Highest Expense: Displays the largest expense with description & date.
  • Zero Spent Days: Shows a list of dates when no expenses were recorded.
  • Income Ratio: Shows your expense-to-income percentage.
  • Updates automatically as you modify your ledger.

Dark Mode

  • Click the Toggle Dark Mode button to switch themes.
  • The app uses CSS variables to adjust colors dynamically.
  • All charts and text adjust properly to improve readability.
